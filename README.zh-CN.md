## 一个入口，五种视角

这一项目集合源于我在**复旦大学张江研究院**约一个月的实习经历。在**叶苏丽老师**指导下，我围绕“AI 如何真正进入房地产行业的业务流程”开展了五个方向不同、服务对象各异的产品探索。

它们并不是五次重复开发：有的帮助开发商提前发现值得关注的区域，有的负责拿地后的规划强排与经济测算，有的进入家庭空间与装修设计，有的服务上海租赁流程，还有一个面向未来银发社区中的老年人、家庭与社区服务团队。

## 五个项目如何连接起来

| 共同原则 | 在项目中的体现 |
|---|---|
| **服务对象不同，但属于同一行业链路** | 覆盖开发商、投拓团队、设计人员、家庭、租客、经纪人、老年人和银发社区运营人员。 |
| **AI 不替代可验证的工程基础** | AI 负责理解、交互与解释；几何、约束、评分、证据和高风险规则仍由明确逻辑控制。 |
| **从模型展示走向真实工作流** | 每个系统都围绕一个具体决策或实际服务过程设计，而不是只展示单一模型能力。 |
| **源码与网站均可独立探索** | 每个项目保留自己的 GitHub 仓库，并提供能够在浏览器中打开的 Render 演示。 |
